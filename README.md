# photonz_ui

world of warcraft addon that includes different useful commands

list of commands:

## photonz_ui_help (show help)
### show help
/photonz_ui_help
### show current settings
/photonz_ui_show 
### enable all settings
/photonz_ui_enable

## maxfps
### show value
/maxfps 
### set value
/maxfps 0-300

## ffxglow
### show value
/ffxglow
### set value
/ffxglow 0-1
