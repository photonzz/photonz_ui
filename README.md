# photonz_ui

world of warcraft addon that includes different useful commands

list of commands:

### show current maxfps
/maxfps 

### set maxfps
/maxfps 0-300

### show ffxglow
/ffxglow

### set ffxglow
/ffxglow 0-1

### photonz_ui_show (show all settings)

/photonz_ui_show

### photonz_ui_enable (enable all settings)

/photonz_ui_enable
