photonz_ui is a lightweight WoW addon that enhances the game with a few convenient features and option settings.

Slash Commands

photonz_ui offers the following slash commands:

## photonz_ui_*
### show help
/photonz_ui_help
### show current settings
/photonz_ui_show 
### enable all settings
/photonz_ui_enable

## maxfps
### show value
/maxfps 
### set value
/maxfps 0-300

## ffxglow
### show value
/ffxglow
### set value
/ffxglow 0-1

## classfriendly
### show value
/classfriendly
### set value
/classfriendly 0-1
